# 🖼️ 添加头像

你需要添加一个头像图片到项目中。

## 要求

- **文件名**: `avatar.png`
- **位置**: 项目根目录（与 index.html 同级）
- **尺寸**: 建议 80x80 像素
- **格式**: PNG（支持透明度）
- **风格**: 像素艺术风格

## 在线生成像素头像

如果需要创建像素风格的头像，可以使用以下工具：

- [Piskel](https://www.piskelapp.com/) - 在线像素编辑器
- [Aseprite](https://www.aseprite.org/) - 专业像素艺术软件（付费）
- [Pixel Art Maker](https://pixelartmaker.com/) - 简单的在线工具
- [Pixilart](https://www.pixilart.com/) - 免费在线编辑器

## 如何添加

1. 将你的头像图片命名为 `avatar.png`
2. 将文件放到 `pixel-blog` 文件夹根目录
3. 刷新浏览器即可看到更新

## 临时方案

如果没有像素头像，可以暂时使用以下方法：

1. **使用纯色方块**: 创建一个简单的 80x80 纯色图片
2. **使用表情符号**: 修改 HTML 中的头像部分为 emoji：
```html
<div class="avatar">
    <span style="font-size: 80px;">🎮</span>
</div>
```
3. **占位图**: 留空或使用默认图标

完成后别忘了提交你的头像到 Git！
